# FactoryFound.ai Vercel Package

This is a Vercel-ready Next.js package for the FactoryFound.ai homepage.

## Run locally

```bash
npm install
npm run dev
```

## Deploy to Vercel

1. Upload this folder to a GitHub repository.
2. In Vercel, create a new project from that repository.
3. Framework preset: Next.js
4. Build command: `next build`
5. Output setting: default
6. Deploy

## Brand settings used

- Dark: `#0a1f35`
- Dark Alt: `#0d2545`
- Accent: `#00aaff`

## Notes

- Font: Poppins via `next/font/google`
- "Factory" is intentionally heavier than "Found.ai"
- This package is a polished landing page shell and can be extended with forms, analytics, CMS content, or app routes
